# PySH
PySH is a python shell which the only library uses is os

---
## Commands
the commands that are in PySH are: 

**quit**: to end the session.

**clear**: to clear the previous inputs and outputs

**help**: to show a help wizard

**printsh**: to print text

**pip**: to install packages with pip
